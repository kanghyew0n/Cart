# 29CM-Frontend-Challenge
29CM 프론트엔드 코딩 과제

<br/>

### 🛠 Install
```
npm install
```

<br/> 

### 📌 Usage
* npm run dev 커맨드를 입력하여 localhost:3000 에서 실행
* npm run server 커맨드를 입력하여 서버 실행 (json server)
    * http://localhost:8000/productItems
    * http://localhost:8000/coupons 
```
npm run dev
npm run server
```
